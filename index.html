Get YOUTUBE Subscriber

1. Get all subscribers

GET http://localhost:3000/subscribers Response with an array of all subscribers details in the database.

2. Get YOUTUBE Subscriber by name

GET http://localhost:3000/subscribers1/names Response with an array of all subscribers with only two fields, their names and subscribed channel.

3. Get YOUTUBE Subscriber by id

GET http://localhost:3000/subscribers/:id Retrieve the details of subscriber with the given ID.
